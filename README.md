# Hi, I'm Addison

**Full-Stack Software Engineer & Game Developer** from the Dominican Republic.

I build practical applications, backend systems, APIs, automation tools, and indie games. I care about clean code, performance, user experience, and understanding how things work behind the scenes.

## What I build

- Full-stack applications with practical interfaces, APIs, auth, databases, and deployment workflows.
- Internal tools, automation, reporting, and business systems that reduce manual work.

- Indie games, prototypes, pixel art, assets, and interactive experiments.
